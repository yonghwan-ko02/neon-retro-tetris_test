01. Platform Overview

세련된 다크 네온 속에 피어난 레트로 아케이드 명작

Neon Arcade는 브라우저만으로 가볍고 생생한 레트로 게임 경험을 선사하는 무의존성 게임 포탈 사이트입니다. 스포티파이의 극도로 정갈한 Achromatic 다크 레이아웃을 기반으로 완성도 높게 제작되었습니다.

Tetris

7-Bag 균등 분배 룰과 SRS 회전 오차 보정을 갖추고 홀드 기능을 융합한 정밀한 클래식 테트리스.

Dino Runner

점프와 슬라이딩 이중 물리 장애물 회피를 도입하고 perspective 원근 배경이 빠르게 가속되는 횡스크롤 러너.

Brick Breaker

패들 충돌부 세분화 기하 반사각을 탑재하고 격파 시 네온 파편 파티클이 사방으로 터지는 타격감의 벽돌깨기.

02. Architecture Map

유기적으로 설계된 바닐라 JS 독립 구조

웹 프레임워크나 복잡한 빌드 도구의 의존성 부하를 걷어낸 **순수 브라우저 API 설계**입니다. 각 컴포넌트는 단방향 제어 흐름 하에 연동됩니다.

  • platform.js: 각 독립 게임의 루프 교대 및 Home 리소스 soft-kill 총괄
  • audio.js: 8비트 주파수를 계산하는 합성 신디사이저
  • LocalStorage: 게임별 스코어 영구 보존

Lobby Navigation Flow

1. Lobby Entry: index.html 로드 및 mute 싱크

2. Game Selection: select-card 클릭 시 active 스위칭

3. Canvas Focus: 즉각 키보드 입력을 위한 자동 포커스

4. Lobby Return: stop() 호출하여 CPU 연산 릴리즈

03. Web Audio Synth

효과음 에셋 제로 - 실시간 8비트 오디오 신디사이징

외부 대용량 오디오 파일을 다운로드하고 프리로드하는 부하와 재생 랙을 제거하기 위해, 웹 브라우저의 내장 소리 발생기인 **Web Audio API**를 사용해 실시간 파형을 연주합니다.

OscillatorNode의 톱니파(sawtooth), 삼각파(triangle)를 조합하고, 잔향을 위해 수학적 지수 볼륨 감쇄(Envelope)를 적용하여 정갈하고 따뜻한 8비트 아날로그 효과음을 선사합니다.

Brick Break Synth Code

playBrickBreak() { // Sharp high frequency noise snap this.playTone([700, 300, 100], 0.12, 'sawtooth', 0.1); } playTone(freqs, duration, type, volume) { const osc = this.ctx.createOscillator(); const gain = this.ctx.createGain(); osc.type = type; // exponential volume decay envelope gain.gain.exponentialRampToValueAtTime(0.0001, now + duration); osc.connect(gain); gain.connect(this.ctx.destination); osc.start(now); }

04. Advanced Physics

조작의 깊이를 선사하는 고해상도 물리 수학식

패들 오프셋 반사각 기하학

공이 패들의 중앙에 맞았을 때는 일직선 위로 가깝게 튕겨 나가고, 가장자리에 비껴 맞을수록 날카로운 수평 각도로 꺾여 가속되도록 충돌 비율 반사 각도 벡터 수식을 구현했습니다.

const hitPoint = (ball.x - (pad.x + pad.w/2)) / (pad.w/2); const bounceAngle = hitPoint * (Math.PI / 3.2); ball.dx = Math.sin(bounceAngle) * currentSpeed; ball.dy = -Math.cos(bounceAngle) * currentSpeed;

네온 파편 입자 시스템 (Shard Emitter)

벽돌 격파 시 생성되는 10여 개의 파편 입자가 중력 가속도를 받으며 흩날리는 Emitter 모듈 설계. 프레임 델타타임 기준 감쇄(decay) 및 즉각 배열 소멸(splice) 처리로 프레임 랙을 방지했습니다.

  • 중력값: p.vy += 0.1
  • 투명감 쇠퇴: p.alpha -= decay

05. CPU Soft-Killing

지속 가능한 플랫폼을 위한 완전한 메모리 릴리즈

단일 웹페이지 상에서 여러 스크린을 숨기고 노출하는 라우팅 구조에서, 이동 후에도 게임 루프가 중단되지 않고 백그라운드에서 CPU 연산을 가하는 심각한 리소스 누수(Memory Leak) 현상이 발생했습니다.

각 게임 코어에 **라이프사이클 stop() 메소드**를 바인딩하여, 로비 복귀와 동시에 cancelAnimationFrame을 걸어 게임 루프 스케줄러를 깨끗하게 강제 제거하고 화면을 리셋함으로써 **CPU 점유율을 0%**로 완벽하게 안정화시켰습니다.

Lobby Screen Return Sequence

  • Lobby Button Click: platform.js 이벤트 가로챔
  • Stop Call: window.NeonBreaker.stop() 등 호출
  • Loop Kill: active flag = false, cancelAnimationFrame 실행
  • Clear Buffer: 캔버스를 깨끗하게 비우고 메트릭스 리셋
